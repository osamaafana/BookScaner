# BookScanner Frontend

React + TypeScript + Vite frontend for the BookScanner application.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- PWA Support

For full project documentation, see the main [README.md](../../README.md) in the project root.
